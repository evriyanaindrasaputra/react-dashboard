### React Dashboard

## Tech
- React
- Tailwind Css 
- Headless UI
- Tanstack Query 
- Tanstack Table
- React Hook Form

## Scripts
- `npm install`: Install Dependencies.
- `npm run dev`: Runs the app in the development mode.

## Setup .env file
- copy from .env.example file
```sh
cp .env.example .env
```
- change API URL with your API 
```sh
VITE_API=<Your API URL>
```
